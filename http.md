
# HTTP Protocol

## Key Notes

 - Stateless Protocol
 - Sits in Application Layer of OSI Networking Reference Layer
 - Runs on top of TCP/IP, default port 80

### HTTP Connections
HTTP Connection is established between Client (IP, Port) and Server (IP, Port).
